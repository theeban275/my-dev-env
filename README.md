# my-dev-env

My developer environment

Prerequisties
1. Vim
2. Ack

Installation Insturctions
1. Copy dot files
```
cp . ~/
```

2. Install Vundle
```
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```

3. Install Vim Plugins
```
vim +BundleInstall +BundleClean +qa
```
